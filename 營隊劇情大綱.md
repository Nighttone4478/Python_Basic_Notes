## 世界觀(背景)
```

在每個人的腦中，都存在一個神秘而龐大的「神經星系」。是一個由「自然科學規律」轉換成能量法則所構成的內在宇宙。神經星系由五種核心能量 AI 維持平衡，它們並不是擬人化情緒，而是象徵人類心智運作的五大能力：

```

1. **光能AI** ——希望、清晰、看見方向的能力
2. **烈能AI** ——行動力、挑戰困難的力量
3. **靈能AI** ——直覺、感官敏銳與冷靜
4. **智能AI** ——邏輯、思考、問題解決
5. **善能AI** ——合作、同理心、善良與關懷

```

這五個 AI 控制著整個神經星系的「核心晶塔」。只要晶塔穩定，人類的思考、情緒、感官和行為就能保持平衡。

```
