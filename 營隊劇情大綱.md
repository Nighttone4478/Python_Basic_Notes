# 寒假國小營隊劇情
### 主題名稱：《核心晶塔：星能裂變計畫》

## 世界觀(背景)
```
在每個人的腦中，都存在一個神秘而龐大的「神經星系」。
是一個由「自然科學規律」轉換成能量法則所構成的內在宇宙。
神經星系由五種核心能量 AI 維持平衡，象徵人類心智運作的五大能力：
```

1. **光能AI** ——希望、清晰、看見方向的能力
2. **烈能AI** ——行動力、挑戰困難的力量
3. **靈能AI** ——直覺、感官敏銳與冷靜
4. **智能AI** ——邏輯、思考、問題解決
5. **善能AI** ——合作、同理心、善良與關懷

```
這五個 AI 控制著整個神經星系的「核心晶塔」。
只要晶塔穩定，人類的思考、情緒、感官和行為就能保持平衡。
```

## 危機起點：裂變風暴
```
某天，一股由壓力、資訊過載、消極念頭混合而成的黑暗能量在星系內暴走。
這股能量形成了名為「失序AI」 的黑影。
失序AI：原本是負責吸收壓力的系統，但因負載過重而「程式異常」。
它誤以為自己被拋棄，爆發後擊碎核心晶塔，五種能量晶片散落在五座星域，
整個神經星系陷入混亂。
```

烈能AI不會因失敗而變弱。

# 科學闖關

## 關卡一
**教案名稱：《摩斯密碼音訊競賽》**
### 【教案成員】
沅亨、柏澔

### 【教案目的】
1. 讓學生了解摩斯密碼的基本結構（短音．、長音－）。
2. 讓學生學會編碼和解碼的過程。
3. 透過「聽、記、譯」三段流程，培養團隊合作與溝通能力。

### 【教案地點】

### 【教案流程】
    3分鐘：開場與玩法介紹
    2分種：分組（兩組，每組 → A/B/C 三隊）
    12分鐘：限時競賽（輪流角色）
    3分鐘：結語 + 得分公布

### 【教案內容】
兩組同時比賽，每組由 6 人分成 A / B / C 三隊（每隊 2 人輪流上場）。
流程如下：

1. A 隊：看到字卡 → 參考表格 → 用蜂鳴器傳送摩斯密碼聲音。
2. B 隊：背對 A 隊 → 只能用「聽」的 → 手寫．與－。
3. C 隊：拿到 B 的符號 → 對照摩斯碼表 → 翻譯成英文字 → 跑去交給關主得分。

限時內分數最高的組獲勝。

### 【分工】
1. 解說2人（遊戲開始時可協助其他人）
2. 計分人員2人

### 【道具】
1. 自製電子電路(按鈕+蜂鳴器+電池) * 2組
2. 紙筆 (讓小朋友記錄摩斯密碼) * 數張
3. 摩斯密碼對照表 * 4張

### 【備註】
1. 注意每個小朋友都要輪到
### 【開場結語】

         A 隊在這裡（指），B 隊在這裡（指），C 隊在這裡（指）。

      要用桌上的摩斯密碼表把字母轉成點和線，
