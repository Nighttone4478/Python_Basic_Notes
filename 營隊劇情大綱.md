## 世界觀(背景)
```
在每個人的腦中，都存在一個神秘而龐大的「神經星系」。

神經星系由五種核心能量 AI 維持平衡，象徵人類心智運作的五大能力：
```

1. **光能AI** ——希望、清晰、看見方向的能力

3. **靈能AI** ——直覺、感官敏銳與冷靜

# 科學闖關

## 關卡一
**教案名稱：《摩斯密碼音訊競賽》**
### 【教案成員】
沅亨、柏澔

### 【教案目的】
1. 讓學生了解摩斯密碼的基本結構（短音．、長音－）。
2. 讓學生學會編碼和解碼的過程。
3. 透過「聽、記、譯」三段流程，培養團隊合作與溝通能力。

### 【教案地點】

### 【教案流程】
    3分鐘：開場與玩法介紹
    2分種：分組（兩組，每組 → A/B/C 三隊）
    12分鐘：限時競賽（輪流角色）
    3分鐘：結語 + 得分公布

### 【教案內容】
兩組同時比賽，每組由 6 人分成 A / B / C 三隊（每隊 2 人輪流上場）。
流程如下：

1. A 隊：看到字卡 → 參考表格 → 用蜂鳴器傳送摩斯密碼聲音。
2. B 隊：背對 A 隊 → 只能用「聽」的 → 手寫．與－。
3. C 隊：拿到 B 的符號 → 對照摩斯碼表 → 翻譯成英文字 → 跑去交給關主得分。

限時內分數最高的組獲勝。

### 【分工】
1. 解說2人（遊戲開始時可協助其他人）
2. 計分人員2人

### 【道具】
1. 自製電子電路(按鈕+蜂鳴器+電池) * 2組
2. 紙筆 (讓小朋友記錄摩斯密碼) * 數張
3. 摩斯密碼對照表 * 4張

### 【備註】
1. 注意每個小朋友都要輪到
### 【開場結語】

#### **開場：**

關主：各位小朋友大家好，我是關主 XXX。

副關主：我是副關主 OOO。

關主：今天我們來到「聲音密碼試煉」！聲音守衛把寶石藏了起來，
      只有用摩斯密碼的「嗶嗶聲」才能跟它溝通。
      你們需要合作，把訊息成功地傳出去、記下來，再翻回英文字母。

副關主：等等每一組同學，請幫我分成 A、B、C 三個小隊。
         A 隊在這裡（指），B 隊在這裡（指），C 隊在這裡（指）。

關主：接著我來說明你們各隊要做的事情喔！

關主：A 隊的同學，你們會拿到一張英文字母卡片，
      要用桌上的摩斯密碼表把字母轉成點和線，
      然後用按鈕按出「短音」和「長音」，
      把聲音訊號傳給 B 隊聽。（走向 B 隊示範）

關主：B 隊的同學，你們要背對 A 隊，只能用「聽」的！
      聽到短音就寫「．」，聽到長音就寫「－」。
      把你們聽到的全部寫在紙上。（走向 C 隊示範）

關主：C 隊的同學，你們要把 B 隊寫的符號，
      再用表格翻回英文字母，
      然後寫到紙上跑來找我對答案，
      答對就能替你們小組得分！（回到原位置）

副關主：等等我們會限時比賽，
         哪一組分數最高就能獲得這一關的獎勵喔！

副關主：那麼，大家準備好了嗎？
         那就讓我們開始吧！


#### **結尾：**

副關主：遊戲結束了，大家剛剛真的超棒喔！先給自己一個大大的掌聲！（拍手）

關主：那麼大家，你們知道剛剛我們用到的「嗶嗶聲」到底是什麼原理嗎？（提問）

關主：好！那我問大家，你們平常用手機、看電視、玩遊戲時，有沒有想過這些機器是怎麼互相傳訊息的呢？（提問）
關主：沒錯，這些訊息其實都是一種「訊號」喔！有的用光、有的用電、有的用網路，而我們今天用的是「聲音訊號」！

關主：那有人知道什麼是訊號嗎？（提問）
關主：舉個例子，當你和朋友講話時，聲音會傳到對方的耳朵；這個「聲音」就是訊號！

副關主：哇～原來我們剛剛是在玩真正的通訊技術耶！好酷喔！
