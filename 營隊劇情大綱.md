# 科學闖關

## 關卡一
**《摩斯密碼音訊競賽》**
### 【教案成員】
沅亨、柏澔

### 【教案目的】
1. 讓學生了解摩斯密碼的基本結構（短音．、長音－）。
2. 讓學生學會編碼和解碼的過程。
3. 透過「聽、記、譯」三段流程，培養團隊合作與溝通能力。

### 【教案地點】

### 【教案流程】
    3分鐘：開場與玩法介紹
    2分種：分組（兩組，每組 → A/B/C 三隊）
    12分鐘：限時競賽（輪流角色）
    3分鐘：結語 + 得分公布

### 【教案內容】
兩組同時比賽，每組由 6 人分成 A / B / C 三隊（每隊 2 人輪流上場）。
流程如下：

1. A 隊：看到字卡 → 參考表格 → 用蜂鳴器傳送摩斯密碼聲音。
2. B 隊：背對 A 隊 → 只能用「聽」的 → 手寫．與－。
3. C 隊：拿到 B 的符號 → 對照摩斯碼表 → 翻譯成英文字 → 跑去交給關主得分。

限時內分數最高的組獲勝。

### 【分工】
1. 解說1人（遊戲開始時可協助其他人）
2. 裁判1人
3. 計分人員2人

### 【道具】
1. 自製電子電路(按鈕+蜂鳴器+電池) * 2組
2. 紙筆 (讓小朋友記錄摩斯密碼) * 數張
3. 摩斯密碼對照表 * 4張
